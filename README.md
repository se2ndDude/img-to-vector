# img-to-vector

A simple web app to convert raster images to SVG vectors using vtracer.

## Installation

```bash
pip install vtracer requests flask
```

## Usage

Run the web app:

```bash
cd /directory/img-to-vector && python main.py
```

Then open http://localhost:5000 in your browser. Upload an image (PNG, JPG, GIF, BMP) and choose color mode. Click "Convert to SVG" to download the vectorized image.

## Requirements

- Python 3.x
- vtracer library
- requests library
- flask library

or whatever
